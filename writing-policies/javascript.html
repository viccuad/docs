JavaScript

JavaScript cannot be converted to WebAssembly, however Javy can execute JavaScript code in a WebAssembly embedded JavaScript runtime, powered by the QuickJS embedded JavaScript engine.

Current State​

Currently there's no Kubewarden SDK for JavaScript. As of the time of writing, Javy is a beta project that was only recently released.
